# 답변

**TCP**와 **UDP**는 **전송 프로토콜**입니다. 데이터를 주고받는 데에 사용됩니다.

### TCP 특징
- 연결지향성 프로토콜이다.
- UDP 보다 비교적 속도가 느리다.
- 데이터가 손상되는 경우 재전송을 통해 데이터의 정확성을 유지하고, 데이터 순서를 보장합니다.
- 혼잡 제어, 흐름 제어를 하여 효율적인 데이터 전송을 할 수 있게 해줍니다.

  
### UDP 특징
- 비연결지향성 프로토콜이다.
- TCP 보다 비교적 속도가 빠르다.
- 데이터가 손상될 수 있으며, 데이터의 손상으로 순서를 보장할 수 없습니다.
- 혼잡 제어, 흐름 제어를 지원하지 않습니다.
